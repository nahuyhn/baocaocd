# CD1 Wireless Presentation Web

Web tĩnh để ôn báo cáo Chuyên đề 1 trên điện thoại.

## Cách dùng GitHub Pages

1. Upload toàn bộ thư mục này lên repository GitHub.
2. Vào Settings → Pages.
3. Chọn Deploy from a branch, branch `main`, folder `/root`.
4. Mở link GitHub Pages để xem trên điện thoại.
